Security & Workspace Settings

Account

Recommended security

  • Enable Google OAuth (SSO) for admins/managers
  • Use long secrets for JWT + session middleware
  • Keep API docs disabled in production
  • Rotate integration tokens regularly
  • Least-privilege roles for agents

This project follows OWASP-style basics: RBAC, rate limiting, strict CSP, and audit logging.
